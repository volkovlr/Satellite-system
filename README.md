# Проект "Satellite system"

В начальный момент времени задаются три числа: количество плоскостей (орбит), по которым летают спутники, количество спутников на одной орбите, расстояние от каждого спутника до Земли. Все орбиты примем круговыми и с равным расстоянием до поверхности Земли.

Задачи:
1) Определить мгновенное покрытие Земли спутниками.
2) За некоторый промежуток времени определить сколько процентов поверхности покроют спутники


Возможные идеи реализации: для каждого спутника будем определять список сот, который он полностью покрывает.

Структуры: можно ввести класс "Спутник", "Орбита". Где-то нужно хранить методы, позволяющие находить процент покрытия.


Предложения: из этого можно сделать библиотеку.
